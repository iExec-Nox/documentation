# Building Confidential Tokens

Welcome to the guide on building confidential tokens with Nox. Learn how to
create ERC-7984 compliant tokens with hidden balances and private transfers.

## Overview

Confidential tokens enable private transactions on public blockchains. With Nox,
you can wrap existing ERC20 tokens or create native confidential assets that
maintain privacy while remaining composable.

## What You'll Learn

- How to create ERC-7984 compliant tokens
- How to wrap existing ERC20 tokens
- How to build token swap applications
- Best practices for confidential token design
